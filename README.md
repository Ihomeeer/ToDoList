# ToDo List
_______________

## Описание
Небольшая ТуДушка.
Стек:
-HTML/SCSS,
-JavaScript

Реализовано добавление пунктов, их удаление, кнопка "очистить", кнопка "забить" (удаляет все пункты на сегодня, добавляет вместо них несколько шутливых), переключение тем (ночь-день).
